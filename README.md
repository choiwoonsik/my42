##### ft_printf 과제

[ft_printf code](https://github.com/choiwoonsik/my42-Seoul/tree/master/42cursus/ft_printf_dir/ft_printf)

1. `make` 입력

2. `cc main.c libftprintf.a` 입력

   (main.c는 테스트 코드, libftprintf는 만든 library code와 printf 코드가 들어있음)
